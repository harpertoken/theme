# Black Theme for VS Code

A minimal black VS Code theme with one-click installation.

## Installation

Run the script to toggle the theme:

```bash
./theme.sh
```

Or use specific commands:

```bash
./theme.sh apply    # Apply theme
./theme.sh undo     # Revert theme
./theme.sh status   # Check current state
```
